# Code Napoléon

Uniform wetboek ingevoerd door [[napoleon-bonaparte|Napoleon]] in 1804. Legde de verworvenheden van de [[franse-revolutie|Revolutie]] vast:
- Gelijkheid voor de wet
- Vrijheid van godsdienst
- Eigendomsrecht

De Code geldt nog steeds in Frankrijk en in vele andere landen. Het was ook een instrument van centralisatie: één wet voor heel Frankrijk, in plaats van een lappendeken van lokale gewoontes.
